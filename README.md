# Anonymous Confession Website

## Overview

Anonymous Confession Website adalah aplikasi web sederhana yang memungkinkan pengguna mengirimkan pesan atau pengakuan secara anonim tanpa perlu membuat akun.
Website ini menekankan kemudahan penggunaan, anonimitas, dan pengalaman interaksi yang sederhana.

---

## Features

### 1. Anonymous Confession

Pengguna dapat mengirimkan pesan secara anonim tanpa perlu login atau registrasi.

### 2. Simple Message Submission

Form sederhana untuk mengirimkan confess dengan cepat.

---

## How It Works

1. Pengguna membuka halaman website.
2. Pengguna menuliskan pesan confession pada form yang tersedia.
3. Pesan dikirim ke server dan disimpan di database.
4. Confess akan ditampilkan dalam daftar confession yang dapat dilihat oleh pengguna lain.
